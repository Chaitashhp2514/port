# Alex Carter — Portfolio

A premium, modern, fully static portfolio site built with **HTML5, CSS3, and Vanilla JavaScript**. No frameworks, no build step.

## Structure
```
portfolio/
├─ index.html
├─ style.css
├─ script.js
├─ assets/
│  └─ resume.pdf   (replace with your real resume)
└─ README.md
```

## Features
- Dark theme by default with light/dark toggle (persisted)
- Sticky glassmorphism navbar
- Animated hero with typing effect & gradient text
- Scroll reveal animations & animated skill bars
- Animated stat counters
- Modern project cards with tech badges
- Timeline experience section
- Validated contact form (frontend only)
- Scroll-to-top button
- Fully responsive (mobile, tablet, desktop)
- SEO-friendly semantic HTML + Open Graph tags

## Run locally
Just open `index.html` in a browser, or:
```bash
npx serve .
```

## Deploy
Drop the folder into **Vercel**, **Netlify**, **Render**, or **GitHub Pages** — no configuration required.
